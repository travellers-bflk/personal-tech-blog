# 版本记录

本分支用于记录博客的功能新增、行为调整和重要改动。

## v0.1.0 - 2026-07-21

### 新增

- 基于 Astro 的静态 Markdown 博客。
- 首页、文章归档、标签页和文章阅读页。
- 两篇首发文章：Vibe Coding 方法论与 AI Agent 工程闭环入门。
- RSS 订阅、sitemap 与 Cloudflare Pages 响应头配置。
- 深色模式切换，用户选择保存在浏览器本地。
- 顶部 GitHub 仓库导航。

### 调整

- 站点名称设为“个人技术笔记”，域名仅用于部署配置。
- 主分支 README 精简为项目概要、内容范围和技术栈。

### 工程与安全

- `.gitignore` 排除环境变量、私钥、日志、缓存、构建产物和本地部署状态。
- Git 提交使用 GitHub noreply 邮箱，避免公开本机个人邮箱。
- 生产构建与类型检查已通过。
